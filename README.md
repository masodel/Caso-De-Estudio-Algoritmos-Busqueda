# Caso de Estudio: Sistemas de Búsqueda para una Biblioteca Digital Estudiantil

## Contexto
La Universidad desea desarrollar un prototipo inicial de un sistema de búsqueda para su futura **Biblioteca Digital Estudiantil**.  
El sistema permite localizar información de manera eficiente dentro de una colección de datos como libros, autores, categorías y códigos.  

Como programadores en formación, se asignó la tarea de investigar, diseñar e implementar algoritmos de búsqueda simples en C#, sentando las bases de este proyecto.

---

## Objetivos
- Aplicar conceptos de búsqueda en estructuras de datos mediante algoritmos implementados manualmente.  
- Fomentar el trabajo colaborativo.  
- Documentar la investigación usando normas APA.

## Funcionalidades del sistema
El sistema desarrollado en C# permite:

1. **Búsqueda lineal** por título de libros.  
2. **Búsqueda binaria** en una lista ordenada de autores.  
3. **Búsqueda del libro más reciente y más antiguo**.  
4. **Búsqueda de coincidencias** dentro de descripciones textuales.  

El sistema es ejecutable desde la consola.

---

## Estructura de ramas en GitHub
- `main`: Rama principal y estable.  
- `dev`: Rama de desarrollo, integración de todas las funciones.  
- `feature/busqueda-lineal`: Implementación de búsqueda lineal.  
- `feature/busqueda-binaria`: Implementación de búsqueda binaria.  

Se realizaron **pull requests** revisados por la pareja y se mantuvo un registro de comentarios y revisiones.
